What is difference between list and set ?

In Python, a list is an ordered collection of elements that can contain duplicates, while a set is an unordered collection of unique elements. This means that a list maintains the order of its elements and allows for duplicates, while a set does not maintain any order and only contains unique elements. Additionally, sets support set-specific operations such as union, intersection, and difference, while lists do not.

Python List comprehension ?

List comprehension is a concise and expressive way to create a new list by iterating over an existing iterable ( e.g., a list, tuple, string) and applying a condition or operation on each element in a single line of code.

squares = [i**2 for i in range(1, 11)]
What is Python Lambda Functions?

In programming, a lambda function is a type of anonymous function that can be defined in a single line of code without requiring a formal name. A lambda function can take any number of arguments, but it must return a value.

For example, the following lambda function takes two arguments and returns their sum:

sum = lambda a, b: a + b
What is Python constructor?

In Python, a constructor is a special method that is called automatically when an object is created. It is used to initialize the instance variables of the object and can be defined using the __init__() method. The constructor is optional, but it is commonly used to set default values for attributes, accept arguments to initialize instance variables, and perform other initialization tasks required for an object to be in a valid state.

Difference between instantanous variables and static variables with examples ?

In Python, there is no concept of "instantaneous" or "static" variables in the way they exist in some other programming languages like Java. However, there are similar concepts of instance variables and class variables.

Instance variables are variables that are unique to each instance of a class, while class variables are shared by all instances of a class. Here's an example to illustrate the difference:

class Car: wheels = 4 # class variable def __init__(self, make, model): self.make = make # instance variable self.model = model # instance variable

In this example, wheels is a class variable because it is defined outside of any method, and it is shared by all instances of the Car class. make and model are instance variables because they are defined inside the __init__ method, and they are unique to each instance of the Car class.

To access a class variable, you can use the class name, like Car.wheels. To access an instance variable, you need to have an instance of the class, like my_car = Car('Honda', 'Civic') and then access the variable with my_car.make.

It's important to note that instance variables can vary from instance to instance, while class variables are shared by all instances. For example, you might have one Car instance with make='Honda' and another Car instance with make='Toyota', but both instances would share the wheels class variable.

What is a Python function and how do you define and call one?

Python constructor is a special type of function that is called when an object is created. The purpose of a constructor is to initialize the object's attributes or properties to a set of initial values.

In Python, the constructor for a class is defined using the __init__ method. This method takes one required argument, self, which refers to the object being created. The __init__ method can also take any number of additional arguments, which can be used to set the initial values of the object's attributes.

For example, here is a simple class definition with a constructor:

class Person: def __init__(self, name, age): self.name = name self.age = age

the __init__ method initializes the name and age attributes of the Person object to the values passed in when the object is created. When a new Person object is created using the syntax person = Person('John', 30), Python automatically calls the constructor to initialize the name and age attributes of the person object.

constructors are optional in Python, and if a class does not define a constructor, Python provides a default constructor that does nothing. However, constructors are often used to ensure that objects are initialized with the correct values before they are used.
