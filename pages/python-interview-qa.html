Python basic QA

What is Python and what is it used for?

Python is a high-level, interpreted programming language that is widely used for web development, scientific computing, data analysis, and artificial intelligence. It is known for its simplicity, readability, and flexibility.

What is a Python module and how do you import one?

A Python module is a file containing Python definitions and statements. You can import a module into your program using the import statement, for example: import module_name. You can also import specific functions or variables from a module using the from keyword, like from module_name import function_name.

What are the different data types in Python and how do you convert between them?

Python has several built-in data types, including integers, floating-point numbers, strings, booleans, and complex numbers. You can convert between these data types using built-in functions such as int(), float(), and str(). For example:

x = '123' y = int(x) # y is now 123 z = str(y) # z is now '123'
What is a Python function and how do you define and call one?

A Python function is a block of code that can be defined and called by a name. To define a function, you use the def keyword, followed by the function name and a set of parentheses containing any arguments that the function takes. For example, the following code defines a function called greet that takes a single argument name:

def greet(name): print("Hello, " + name + "!") greet("John")

To call a function, you simply use its name followed by a set of parentheses containing any arguments that the function requires. For example, to call the greet function defined above, you would write above greet('John')

Types Of Inhertance on python?

Single Inheritance: In single inheritance, a subclass is derived from a single base class. The subclass inherits all the attributes and methods of the base class.

class Animal: def __init__(self, name): self.name = name def make_sound(self): pass class Dog(Animal): def make_sound(self): return "Woof!" my_dog = Dog("Buddy") print(my_dog.name) # Output: Buddy print(my_dog.make_sound()) # Output: Woof!

In this example, the Dog class is derived from the Animal base class using single inheritance. The Dog class inherits the name attribute and the make_sound() method from the Animal class.

Multiple Inheritance: In multiple inheritance, a subclass is derived from two or more base classes. The subclass inherits all the attributes and methods of all the base classes.

class ElectricCar: def __init__(self, make, model): self.make = make self.model = model def start_engine(self): pass class ElectricVehicle: def charge_battery(self): pass class ElectricCar(Car,ElectricVehicle): def __init__(self, make, model): super().__init__(make, model) my_car = ElectricCar("Tesla", "Model S") print(my_car.make) # Output: Tesla my_car.charge_battery()

In this example, the ElectricCar class is derived from both the Car and ElectricVehicle classes using multiple inheritance. The ElectricCar class inherits the make, model, and start_engine() attributes and methods from the Car class, as well as the charge_battery() method from the ElectricVehicle class.

Multilevel Inheritance: In multilevel inheritance, a subclass is derived from a base class, which in turn is derived from another base class. The subclass inherits all the attributes and methods of both the base classes.

class Vehicle: def __init__(self, make, model): self.make = make self.model = model def start_engine(self): pass class Car(Vehicle): def drive(self): pass class SportsCar(Car,ElectricVehicle): def go_fast(self): pass my_car = SportsCar("Porsche", "911") print(my_car.make) # Output: Porsche my_car.start_engine() # This method is inherited from Vehicle

In this example, the SportsCar class is derived from the Car class, which is in turn derived from the Vehicle class. The SportsCar class inherits the make, model, start_engine(), and drive() attributes and methods from the Car class, as well as the go_fast() method defined in the SportsCar class.

Hierarchical Inheritance: In hierarchical inheritance, multiple subclasses are derived from a single base class. Each subclass inherits all the attributes and methods of the base class.

class Animal: def __init__(self, name): self.name = name def make_sound(self): pass class Cat(Animal): def make_sound(self): return "Meow!" class Dog(Animal): def make_sound(self): return "Woof!" my_cat = Cat("Fluffy") my_dog = Dog("Buddy") print(my_cat.name) # Output: Fluffy print(my_cat.make_sound()) # Output: Meow! print(my_dog.name) # Output: Buddy print(my_dog.make_sound()) # Output: Woof!

In this example, both the Cat and Dog classes are derived from the Animal base class using hierarchical inheritance. Each subclass inherits the name attribute and the make_sound() method from the Animal class.
