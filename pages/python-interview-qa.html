Python basic QA

What is Python and what is it used for?

  • Python is a high-level, interpreted programming language that is widely used for web development, scientific computing, data analysis, and artificial intelligence. It is known for its simplicity, readability, and flexibility.

What is a Python module and how do you import one?

  • A Python module is a file containing Python definitions and statements. You can import a module into your program using the import statement, for example: import module_name. You can also import specific functions or variables from a module using the from keyword, like from module_name import function_name.

What are the different data types in Python and how do you convert between them?

  • Python has several built-in data types, including integers, floating-point numbers, strings, booleans, and complex numbers. You can convert between these data types using built-in functions such as int(), float(), and str(). For example:
  • x = '123' y = int(x) # y is now 123 z = str(y) # z is now '123'

What is a Python function and how do you define and call one?

  • A Python function is a block of code that can be defined and called by a name. To define a function, you use the def keyword, followed by the function name and a set of parentheses containing any arguments that the function takes. For example, the following code defines a function called greet that takes a single argument name:
  • def greet(name):    print("Hello, " + name + "!") greet("John")
  • To call a function, you simply use its name followed by a set of parentheses containing any arguments that the function requires. For example, to call the greet function defined above, you would write above greet('John')
